context

passwordless ssh is very useful, but to be a little more secure you usually want a passphrase on your private key. but it gets annoying to have to type in the passphrase everytime... enter ssh-agent.

however, apparently it's a bit of a trick to get it to work properly if you want to initialize it in .bashrc or the like.. Below is a bash script that will initialize things properly, and then re-use the existing ssh-agent if it's already running. As the comments in the script suggest, I got most of this somewwhere else, but had to modify it slightly to make it work for me...

Generate an anisotropic, non-centered, non-axis aligned mulitivariate Gaussian data set

context

Sometimes it's useful to have a non-trivial but predictable random data set on which to test things. It took a minute to put this together from various sources, so I thought I'd put it here for general use.

Among other things, this example includes how to

  • Properly generate a multivariate sample by using the Cholesky decomposition of the covariance instead of the covariance.
  • How to generate a random d-dimensional rotation matrix.
