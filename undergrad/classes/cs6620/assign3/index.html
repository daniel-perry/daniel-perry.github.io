Design Decisions:
    For this assignment I've kept the same basic structure of my ray tracer, which consists of:
Base Classes

  • vector3d
  • hpoint
  • ray
  • image
Objects
  • Object
    • Group
    • Primitive
      • Sphere
      • AxisBox
      • Triangle
      • Disk
      • Ring
      • HeightField
Other
The main loop of the array is in the same .cc file as the main() function. Scene description is still being done as object declarations in that same file, since I have not yet implemented a scene description parser or the like...
A normal scene will consiste of a Group object, to which different Primitive subclasses are added. A std::vector or lights is created to hold the different lights, and is looped over in the main tracing loop.
This design was chosen so as to allow more time to spend on the assignment requirements, though later on I would like to implement a scene description parser and implement a World class to which I can add everything else, so that the code looks a little more neat.
