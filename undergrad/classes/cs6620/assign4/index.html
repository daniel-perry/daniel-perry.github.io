Design Decisions:
    For this assignment, I implemented 4 new Material classes, PhongMaterial, MetalMaterial, DielectricMaterial, and CoupledMaterial. I was able to keep using the basic infrastructure already established with some changes this time.
    Notably, several interfaces were overloaded to match the interface used in the req.h files (so that I can include them without having to change it). This also required some typedef's so that my names like vector3d and rgb are understood as Vector and Color. This made sense since now my interfaces match those given in assignments, whereas before they didn't (which is a pain). I guess I'm a conformist.
    I had some trouble getting the transparent material working for a while there, because my box intersection code wasn't correct... somehow it slipped through the other assignments.
    For the reflection and transparent materials, I used schlicks approximation, for the obvious speed advantage, and they seem to look pretty good. For the transparent material, I was getting really slow render times (about 39 seconds for the required image). After implementing the "ray trimming" that time dropped considerably (for the required image, down to 4 seconds). Althought still not ideal, it is at least reasonable. If I have more time later on, I'd like to optimize more to get my rendering time down.
    This assignment seemed considerably less painfull than past assignments, though seemed to make the images appear much better. Cool stuff...
